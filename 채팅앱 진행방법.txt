1. firebase setting
 - 구글에서 firebase로 접속(console.firebase.google.com)
 - 로그인 후, 새로 만들기 추가
 - 새 프로젝트 추가하여 naming에 자유롭게 작성
 - 구글 애널리틱스를 선택하지 않음으로 선택하고 진행버튼 클릭
 - 프로젝트 생성 후, 로딩이 끝나면 중앙에 위치한 </>(웹)을 클릭
 - 동일한 이름으로 작성 후, 진행 버튼 클릭
 - [Firebase SDK 추가]에서 소스코드 복사하여 작업창에 추가 + 하단의 <콘솔로 이동> 버튼 클릭
 - 콘솔 창에서 좌측 메뉴 중 Database 클릭 후, 변경된 창에서 하단에 위치한 Realtime Database에서 <데이터 베이스 만들기> 버튼 클릭
 - [실시간 데이터베이스 보안 규칙] 팝업창에서 테스트 모드로 시작(라디오 버튼)을 선택

2. firebase App과 Database 포함하기
 - 작업창에 추가한 스크립트 코드 중 <script src="https://www.gstatic.com/firebasejs/8.2.0/firebase-app.js"></script> 를 복사한 후, 하단에 <script src="https://www.gstatic.com/firebasejs/8.2.0/firebase-database.js"></script>로 붙여넣기 및 뒷 부분 firebase-app.js -> firebase-database.js로 변경
 - form 및 입력요소에 따른 작업 코드 추가
